Reward Function — Rmin ตาม paper Eq.6
Rmin(at) = mini( ERi(st, at) / Einit )
ERi = energy เหลือของ node i หลัง simulate round นั้น · Einit = 5.0 J · min = เอา node ที่แย่สุด · ค่าอยู่ระหว่าง 0.0 (หมด) ถึง 1.0 (เต็ม)
Rmin ตลอด Simulation
เส้นบางจาง = ค่าจริงทุก round · เส้นทึบ = moving average 50 rounds

Q-Update ทีละรอบ — Eq.1 ใน paper
สูตรที่ใช้อัปเดต Q-table ทุกรอบ
Q(st, at) = (1 − α) × Q(st−1, at−1) + α × [ Rmin(at) + γ × maxAS Q(st+1, at+1) ]
α = 0.5 (learning rate) · γ = 0.5 (discount factor)
ตัวอย่าง: Q เดิม=0.45, Rmin=0.47, max Q ถัดไป=0.52
Q ใหม่ = 0.5×0.45 + 0.5×(0.47 + 0.5×0.52) = 0.225 + 0.5×0.73 = 0.590
